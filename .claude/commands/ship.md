テストを通してmainにマージ・デプロイするまでの一連の流れを実行します。

## 手順

1. `cd app && npm run lint` でlintエラーがないか確認する
2. `npm test` で全テストがPASSか確認する
3. `git status` で変更内容を確認してコミットメッセージの案を提示する
4. ユーザーの承認を得てからコミット・プッシュする
5. PRが必要かどうか確認する（featureブランチならPR作成を提案）

## 注意

- テストが1件でもFAILしていたらデプロイを止める
- lintエラーがあれば先に修正する
- コミットメッセージはユーザーに確認してから実行する
